# React Boilerplate SSR

> Universal React v16.0 boilerplate

## Getting Started

### Client Side Development

```bash
cd client
yarn install
yarn dev
```

Navigate to `http://localhost:8080`

### Server Side Development

```bash
cd client
yarn install
yarn run dev:server

cd server
yarn install
yarn watch
```

Navigate to `http://localhost:3000`
